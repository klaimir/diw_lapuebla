#Icono de Hamburguesa:

El elemento con clase hamburger es el ícono que el usuario hace clic para mostrar u ocultar el menú. Se llama a la función toggleMenu() cuando se hace clic en este icono.

#Menú Hamburguesa (#menu-hamburguesa):

El menú hamburguesa está inicialmente oculto (display: none). Este menú aparecerá cuando el usuario haga clic en el icono.

#JavaScript (toggleMenu() function):

La función toggleMenu() usa JavaScript para alternar la propiedad display del menú hamburguesa.
Si el menú está visible (display: block), la función lo oculta (display: none).
Si está oculto, la función lo muestra (display: block).
